# วิธีการเปิดเว็บ (Localhost) สำหรับทำงานที่ออฟฟิศ

เนื่องจากระบบเซอร์วิส Apache (Web Server) ที่เครื่องออฟฟิศอาจจะไม่ได้ถูกตั้งให้รันอัตโนมัติ เวลาที่คุณต้องการแก้ไขโค้ดและดูผลลัพธ์ผ่าน localhost ให้ทำตามขั้นตอนนี้ครับ:

## วิธีรัน Local Server ด้วย PHP

1. เปิดโปรแกรม **PowerShell** หรือ **Command Prompt (cmd)**
2. พิมพ์คำสั่งด้านล่างนี้ลงไป แล้วกด Enter:
   ```cmd
   c:\xampp\php\php.exe -S localhost:8080 -t c:\qc
   ```
3. เมื่อระบบแสดงข้อความว่า `Development Server (http://localhost:8080) started` แสดงว่าเปิดเซิร์ฟเวอร์สำเร็จ
4. เปิดเบราว์เซอร์ แล้วเข้าลิงก์:
   👉 **http://localhost:8080**

### ⚠️ ข้อควรระวังสำคัญ
- **ห้ามปิดหน้าต่างสีดำ (PowerShell/cmd)** ที่คุณใช้รันคำสั่งเด็ดขาด! ให้ใช้วิธี **ย่อหน้าต่าง (Minimize)** แทนครับ
- หากเผลอปิดหน้าต่างไป ตัวเว็บไซต์จะเข้าไม่ได้ (Connection Refused) ให้กลับไปเปิด PowerShell แล้วพิมพ์คำสั่งในข้อ 2. ใหม่อีกครั้งครับ

---

## กรณีที่ Port 8080 ใช้งานไม่ได้ (เข้าแล้วหมุนค้าง/Error)
หากระบบแจ้งว่า 8080 ถูกใช้งานไปแล้ว คุณสามารถเปลี่ยนเป็นเลขพอร์ตอื่นได้เลย เช่น `8081` หรือ `8888` ยกตัวอย่างเช่น:

```cmd
c:\xampp\php\php.exe -S localhost:8888 -t c:\qc
```
แล้วคุณก็เข้าเว็บผ่าน URL: **http://localhost:8888** แทนครับ
